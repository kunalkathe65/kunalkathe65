Railsafar

Railsafar is a MEAN stack application developed during my internship. The application allows users to book train tickets, search for trains, get seat availability etc. It has been deployed on two platforms : Heroku & AWS EC2 instance on Nginx Server. Project Link : Raisafar

Github Finder

Github Finder is a simple React JS application to search for users on Github. It then shows various info like user profile, public repos etc. using Github API. It has been deployed on Heroku. Project Link : Github Finder

Contact Keeper is MERN stack application which is basically a contact manager to add, delete & update contacts. JWT is used for user authentication. Mongo Atlas is used as a database service. It has been deployed to Heroku. Project Link : Contact Manager Project
